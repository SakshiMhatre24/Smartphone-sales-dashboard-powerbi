# Smartphone Sales Dashboard (Power BI)

Interactive Power BI dashboard analyzing smartphone sales performance, customer behavior, and payment trends.

## Key Metrics
- Total Sales, Transactions, Quantity
- Average Order Value and Unit Price

## Analysis Includes
- Monthly sales trends
- City-wise sales distribution
- Brand and model performance
- Payment method analysis
- Customer ratings by age group

## Tools Used
- Power BI (Power Query, DAX, interactive dashboards)
